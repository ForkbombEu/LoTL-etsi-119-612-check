- `xmllint` is not installed in the current development environment. Command-boundary success/failure/catalog behavior is covered with deterministic mocked runners; the real local assessment reports `unsupported` as designed.
- Schema validity is only Annex A.2.1 binding evidence. The normative document prevails over known schema conflicts.
- XPath expressions are not available from `xmllint`; diagnostics include its line/column and message evidence.
- The TS 119 612 alternative XML binding and Annex A.2.2/Table A.1 mapping remain unimplemented.
